# TankaPlay

## Original

[ProsePlay](https://github.com/proseplay/proseplay.git)

## Usage

Install:

```
npm install
```

Run:

```
npm start
```

Build:

```
npm run build
```

To develop with a local version of ProsePlay:

```
cd <path/to/proseplay>
npm link
cd <path/to/proseplay.net>
npm link proseplay
```
